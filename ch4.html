Chapter 4

Communicating in Text Channels

Text channels are at the core of Discord communication. To use them effectively, you should know how to manage your messages, send new ones, reply to specific messages, and read messages from others.

What Is a Text Channel?

A text channel can be identified in the channel list by the # symbol beside their name. Text channels are the place where standard daily communication takes place. Server members can send text messages and make threaded conversations, as well as send files and images depending on the server's rules and permission set up. Text channels retain a permanent history of messages unless deleted.

ℹ️
Text vs. Voice

Text channels use a # symbol. Voice channels use a speaker icon.

Reading Messages

The main chat window displays the conversation of whichever channel you've selected on the left-hand channel list panel. As mentioned earlier, messages are displayed chronologically, with the new messages appearing from the bottom. If you scroll too far up in the chat history, select the "Jump to Present" at the bottom of your screen to return.

The "Jump to Present" bar allows you to quickly return to the latest messages.

If new messages were sent since your last time opening a channel, a red line in the chat will signify where you left off. Messages are tied to user profiles, and contain a time stamp.

The red divider signifies where you left off, with timestamps shown for each message.

Sending a Message

1

Ensure you have permission to type in the channel you have selected. If you do, a clickable input area will be visible at the bottom of your screen.

2

Type your message in the input area.

The message input bar at the bottom of a text channel.
3

Press Enter to send the message.

⌨️
New Line vs. Send

To avoid accidentally sending a message when you are attempting to add a new line to your message, use Shift+Enter instead. Adding new lines improves readability on longer messages.

Replying to a Specific Message

Channels can become hectic, and a reply to a specific user can get lost when other users are also sending messages. The Reply feature links your message with one previously sent, quoting the prior message on your own. A reply to your message will be highlighted and trigger a notification.

1

Locate the message you want to reply to and move your mouse over it, revealing small action icons on the right.

Hovering over a message reveals action icons like the Reply arrow.
2

Click the reply arrow icon. The message input bar will show a preview of the message you are replying to.

3

Type your response in the input bar and press Enter to send it.

Editing and Deleting Your Messages

You can edit or delete any message you have sent at any time.

Editing a Message

1

Hover over your message and click the three-dot menu that appears.

Opening the three-dot "More" menu on your message to edit it.
2

Click Edit Message. The message will become editable in place.

The message context window with the "Edit Message" button highlighted.
3

Make your changes, then press Enter to save or Escape to cancel.

Editing messages is not private. Any edited message will permanently retain an (edited) label. However, other users are unable to see the original message.

Deleting a Message

ℹ️
Note

You can only delete messages sent by you. Administrators can delete anyone's messages.

1

Place your cursor over the intended message and click the three-dot button.

Locating the Delete Message option in the message's "More" menu.
2

Click the red "Delete Message" option near the bottom of the newly opened menu.

Selecting the red "Delete Message" button.
3

A popup dialog box will appear. Review that you have selected the correct message and click to confirm the deletion.

The confirmation dialog before permanently deleting a message.

Adding Emoji Reactions

Reactions are great for acknowledging a message without sending a full reply.

1

Hover over a message to reveal the action icons.

2

Click the smiley face icon on the right.

The "Add Reaction" button located in the message action bar.
3

Search for an emoji. Click the emoji to react to the message.

Selecting a specific emoji to react to a message.
4

You'll now see your reaction under the message. Click it once to remove it.

An emoji reaction appearing underneath a sent message.

Uploading Files and Images

Some servers allow members to share images, documents, or other files in text channels. This depends on the permissions granted to your assigned role.

📎
Tip - Drag and Drop

You can skip the steps below entirely by dragging any file directly into your Discord window to automatically attach it to your message. If you prefer this method, drag your file now. Otherwise, follow the steps below.

1

Verify you have permission: In the message box click the (+) Icon. An options menu opens and a button called "Upload a File" is revealed to users who have permission.

The plus (+) icon menu for uploading files and images.
2

Click "Upload a File." Your system's file explorer opens automatically.

3

Pick whatever image, file, or document you'd like to send.

4

Write a message in the input bar or keep it blank. Press Enter to send.

📁
File Size Limit

Discord is not optimal for sharing large files. Instead, use an online file sharing platform like Dropbox or Google Drive.

Using @Mentions

Busy channels lead to lost messages. @Mentions solve this by highlighting your message to a specific user and sending them a notification.

1

Use an @ symbol in your message followed by the beginning of their username or display name. A dropdown list of users in the server whose names begin with those characters appears.

The @mention dropdown list for selecting a specific server member.
2

Continue typing the user's name until the dropdown list is narrowed down enough to locate the user. Click the user to autofill your mention.

An autofilled @mention appearing highlighted in the message box.
3

Conclude your message and press Enter to send. The mentioned user will receive a notification for your message.

📢
Good to Know: @Mention Etiquette

With the right permissions, you may also be able to use @here to notify everyone online, or @everyone to notify all members. Avoid abusing this feature as it can create alert fatigue and cause users to mute the server completely.
