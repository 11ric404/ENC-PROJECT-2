Understanding Discord's layout
Chapter 2

Navigating the Discord Interface

Before you begin using Discord confidently, you should know what every panel or button does. This guided tour covers the Discord desktop layout.

Overview of the Main Layout

Most of your time using Discord will be spent in servers. Discord's iconic server interface is divided into several distinct panels. Before joining a server in Chapter 3, you should familiarize yourself with these panels.

Panel 1: The Server List (Far-Left Sidebar)

On the far left, you'll notice a narrow column. This is your server list. It contains circular icons allowing you to switch between servers. Think of these like bookmarks for different communities. Since you haven't joined any, you'll only see some default buttons.

The Server List panel showing joined communities.
  • Discord logo button located at the top will lead you to Home Screen (direct messages).
  • Every circular button underneath represents the server that you are a part of. Click on any of the buttons to access that server.
  • The Green + button at the bottom lets you create or join a server.
  • If there is a red notification mark next to the server icon, then you have unread messages in that server.

Panel 2: The Channel List (Second Panel)

Clicking on the server icon opens up the second window where you will see all the channels available in the particular server.

The Channel List panel for a specific server.
  • The channels are categorized into expandable/contractible groups (represented by the gray header in each category).
  • If there is a "#" symbol, then it represents the text channel for text-based communication.
  • If there is a speaker symbol, then it is the voice channel.
  • You can use the arrow next to the category name to expand or contract that channel group.
  • The server name is listed at the top of this window.

Panel 3: The Main Chat Window (Center)

This is the largest area which represents the workspace. It changes contents based on which channel is selected.

The Main Chat Window where messages are read and sent.
  • When in text channels, it contains chat messages ordered chronologically (most recent messages shown near the bottom of your screen).
  • The message input box is found at the very bottom where you can write and submit messages.
  • At the top, the name of the channel is displayed, together with its description.

Panel 4: The Members Panel (Right Sidebar)

This sidebar lists all the users in the selected server.

The Members Panel showing users currently in the server.
  • The members are classified according to their roles. In academic servers, you may see roles like instructor, TA, and student.
  • A green dot next to the name indicates that the user is currently active.
  • A yellow half circle indicates that the user is inactive (idle).
  • A gray dot next to the name means the user is offline/invisible.
  • By clicking on any of these names, you will see the user's profile and be able to send them a direct message.
💡
Tip

You can easily collapse the members panel by clicking the person symbol in the toolbar.

The User Controls Bar

Your personal control center is located on the bottom left of your window. It contains your profile photo, your display name, as well as a few control icons tied to you.

The User Controls bar for managing your microphone, headphones, and settings.

Microphone Button: When this icon is gray, you are unmuted. If you are in a voice channel, your microphone will transmit audio. To mute yourself, click the icon. A red slash will appear on the microphone, indicating others cannot hear you.

🎤
Best Practice

Similar to Zoom or other communication platforms you may be familiar with, muting yourself unless you are currently speaking helps reduce background noise for others in the channel.

Headphone Button: When this icon is gray, you can hear others. If you are in a voice channel, and others speak, your audio output device will transmit that. To deafen yourself, click the headphone icon. A red slash will appear on the headphone, as well as the microphone automatically. Think of this as a reverse mute.

ℹ️

Audio Controls are covered in detail in Chapter 6 Voice Channels

Settings Gear: Clicking the gear icon opens User Settings, where you can manage your profile, notification preferences, audio devices, privacy settings, and appearance. Settings are covered in detail in Chapter 7.

The Search Bar

The search bar, located in the top right of your application window, is a powerful tool that allows you to find old messages. You can filter your search with a variety of options, such as by keyword, date, sender, or attachment.

The search bar for finding specific messages.

For even faster navigation to specific groups, channels, servers, or DMs, press Ctrl + K (Windows) or ⌘ + K (Mac) to open the Quick Switcher. Think of this as a search bar that helps you find locations, rather than specific messages. This feature becomes increasingly helpful as your server list grows.

The Quick Switcher for jumping between channels and servers.

Navigating to the Home Screen

At any point, you can click the Discord logo at the top of your server list to navigate out of a server and back to Discord's main home screen. On the Home screen you will find:

  • Your direct messages with other users.
  • Your Friends list: A red circle with a number indicates a pending friend request.
  • Message requests: A red circle with a number indicates a pending message request.
The Discord logo button used to navigate back to the Home screen.
